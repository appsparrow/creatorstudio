## 📊 DECISION MATRIX

### BEAUTY PRODUCTS

#### Skincare - Face (Serums, Moisturizers, SPF)
```
Hook Format:     Price Reveal OR Discovery
Content Format:  Product Demo (15-20s)
Setting:         Bathroom/Vanity
Key Script:      "It's [ingredient], costs $[X], I've been using for [timeframe]"
Example:         "I spent $23 on this serum and threw out my $80 one"
Why It Works:    Skincare buyers want proof of results + ingredients
```

#### Skincare - Body (Lotions, Scrubs, Self-Tanner)
```
Hook Format:     Before/After OR Discovery
Content Format:  Product Demo (20-30s)
Setting:         Bathroom
Key Script:      "I used this for [X days]. Here's what happened."
Example:         "I fake tanned with this $12 mousse. Day 3 results."
Why It Works:    Body products need visual proof, longer demo time
```

#### Makeup - Face (Foundation, Concealer, Powder)
```
Hook Format:     Price Reveal OR Comparison
Content Format:  Product Demo (15-20s)
Setting:         Bathroom/Vanity
Key Script:      "This is [product] in [shade]. It [specific benefit]. I'm wearing it now."
Example:         "This $15 foundation doesn't oxidize. Wearing it since 7am."
Why It Works:    Face makeup = universal need, price matters, shade match critical
```

#### Makeup - Eyes/Lips (Mascara, Liner, Lipstick)
```
Hook Format:     POV OR Social Proof
Content Format:  Product Demo (15s)
Setting:         Bathroom/Natural light
Key Script:      "POV: You found [product type] that actually [benefit]"
Example:         "POV: You found a $9 mascara that doesn't flake by noon"
Why It Works:    Eye/lip products = specific pain points, short demo needed
```

#### Tools/Gadgets (Brushes, Gua Sha, LED Masks)
```
Hook Format:     Discovery OR Opinion
Content Format:  Product Demo with technique (20-30s)
Setting:         Bathroom
Key Script:      "Why is no one talking about [feature]? Here's how I use it."
Example:         "This $18 face roller has a textured side. Game changer."
Why It Works:    Tools need education, buyers want to see it in action
```

### FASHION PRODUCTS

#### Clothing - Dresses
```
Hook Format:     Price Reveal OR POV
Content Format:  Try-On (20-30s)
Setting:         Bedroom/Mirror
Key Script:      "I spent $[X] on this dress. It's [fabric], runs [sizing], [occasions]."
Example:         "I spent $28 on this dress and wore it to 3 weddings"
Why It Works:    Dresses = occasion-based, fit and fabric matter most
```

#### Clothing - Jeans/Pants
```
Hook Format:     POV (always)
Content Format:  Try-On (20-30s)
Setting:         Bedroom/Full-length mirror
Key Script:      "POV: You finally found [pants type] that fit [body concern]"
Example:         "POV: You found jeans that fit your waist AND hips"
Why It Works:    Jeans = universal fit struggle, POV triggers instant recognition
```

#### Clothing - Tops/Basics
```
Hook Format:     List Format OR Social Proof
Content Format:  Multiple ways to style (20-30s)
Setting:         Bedroom
Key Script:      "3 ways to style this $[X] [item]. [Occasion 1], [2], [3]."
Example:         "3 ways to wear this $15 bodysuit: work, date, brunch"
Why It Works:    Basic tops = versatility sells, show multiple looks fast
```

#### Shoes
```
Hook Format:     Price Reveal OR Comparison
Content Format:  Try-On + Walk Test (20s)
Setting:         Bedroom → Outdoor (if possible)
Key Script:      "These are $[X]. [Brand] dupe. [Comfort note]. Been wearing [timeframe]."
Example:         "These $40 sandals are Golden Goose dupes. Worn for 3 months."
Why It Works:    Shoes = comfort + dupe market huge, need walking proof
```

#### Accessories (Bags, Jewelry, Hats)
```
Hook Format:     Social Proof OR Opinion
Content Format:  Styling Demo (15-20s)
Setting:         Outdoor/Coffee shop/Natural light
Key Script:      "I've seen this [item] everywhere. Here's why it's worth it."
Example:         "This $22 crossbody is all over Paris. Here's why."
Why It Works:    Accessories = trend-driven, social proof = FOMO trigger
```

## 🎬 CONTENT FORMAT DEEP DIVE

### When to Use Each Format

**Format A: Product Demo (15-20s)**
- **Use for:** Single-product focus, feature-heavy items
- **Products:** Makeup, skincare, small gadgets
- **Structure:** Hook → Demo → Reaction → CTA
- **Strength:** Fast, repeatable, easy to batch

**Format B: Try-On (20-30s)**
- **Use for:** Fit/sizing critical products
- **Products:** Clothing, shoes, accessories worn on body
- **Structure:** Hook → Before → Trying on → Result → CTA
- **Strength:** Builds trust through visual proof

**Format C: Comparison (15-20s)**
- **Use for:** Dupes, before/after, alternatives
- **Products:** Any with direct competitor
- **Structure:** Hook → Side-by-side → Key difference → CTA
- **Strength:** Leverages existing brand awareness

**Format D: Storytime UGC (20-30s)**
- **Use for:** Problem-solution products, transformations
- **Products:** Wellness, organization, problem-specific tools
- **Structure:** Hook → Problem → Using product → Result → CTA
- **Strength:** Emotional connection, relatable journey

**Format E: List (20-30s)**
- **Use for:** Versatile products, multi-benefit items
- **Products:** Basics, accessories, multi-use tools
- **Structure:** Hook → Point 1 → Point 2 → Point 3 → CTA
- **Strength:** High watch time, information-dense

## ⚡ QUICK PICK SHORTCUTS

### By Product Price Point

**Under $20 — Use Price Reveal Hook**
- Triggers impulse buy
- "I spent $X on this" = instant curiosity
- Works for: Drugstore beauty, fast fashion, Amazon finds

**$20-$50 — Use POV or Discovery Hook**
- Mid-range = needs justification
- POV creates relatability
- Discovery highlights unique feature
- Works for: Mid-tier fashion, specialty beauty, curated home

**Over $50 — Use Comparison or Opinion Hook**
- Higher price = need social proof
- Compare to luxury alternative
- Position as investment piece
- Works for: Designer dupes, premium skincare, quality basics

---

### By Product Maturity

**Trending/Viral Product — Use Social Proof Hook**
- "Everyone's buying this. Here's the real review."
- Ride the existing wave
- Post quickly while trending
- Works for: TikTok viral items, influencer collabs

**Established Product — Use Discovery or Opinion Hook**
- "Why is no one talking about [feature]?"
- Find new angle on known product
- Works for: Classics, staples, underrated favorites

**New Launch — Use Before/After or Price Reveal**
- No existing reviews = need proof
- Show results or value proposition
- Works for: New brand launches, first-to-market products

---

### By Audience Intent

**High Purchase Intent (Bottom of Funnel)**
- Use: Product Demo, Try-On
- Viewer wants: Specifics (shade, size, fit, durability)
- Script focus: Details, proof, direct CTA
- Example: "[Product] in [shade/size]. It's [fabric/formula]. $[X]. Link below."

**Discovery/Research (Middle of Funnel)**
- Use: Comparison, List Format
- Viewer wants: Options, education
- Script focus: Benefits, alternatives, why this one
- Example: "3 reasons this beats the viral version. Number 2 surprised me."

**Awareness (Top of Funnel)**
- Use: POV, Social Proof
- Viewer wants: Relatability, recognition
- Script focus: Problem identification, FOMO
- Example: "POV: You've been gatekeeping this $18 find for months"

## 🎯 HOOK SELECTION FLOWCHART

```
START: What's the main selling point?

├─ PRICE (It's affordable/great value)
│  └─ Use: PRICE REVEAL HOOK
│     "I spent $X on this and [outcome]"
│
├─ PROBLEM SOLUTION (It fixes a specific issue)
│  └─ Use: POV HOOK
│     "POV: You finally found [solution to relatable problem]"
│
├─ HIDDEN FEATURE (It has an unknown benefit)
│  └─ Use: DISCOVERY HOOK
│     "Wait, why is no one talking about [feature]?"
│
├─ TRENDING (Lots of people buying it)
│  └─ Use: SOCIAL PROOF HOOK
│     "I've seen [X people/places] using this. Here's why."
│
├─ CONTRARIAN (Goes against popular opinion)
│  └─ Use: OPINION/TWIST HOOK
│     "Everyone says [X]. They're wrong. Here's why."
│
├─ TRANSFORMATION (It creates visible change)
│  └─ Use: BEFORE/AFTER HOOK
│     "I used this for [timeframe]. Here's what changed."
│
└─ ALTERNATIVE (It's better than known product)
   └─ Use: COMPARISON HOOK
      "[Expensive brand] vs this $X version. The difference is wild."
```

## 💡 ADVANCED STRATEGY NOTES

### Product Combo Patterns (Week 1 Strategy)

**Day 1-2: Foundation Videos (Build Trust)**
- Product 1: Foundation (Price Reveal)
- Product 2: Concealer (Product Demo)
- Why: Face makeup = broad appeal, easy entry point
- Audience: General beauty buyers

**Day 3-4: Niche Down**
- Product 3: Lip oil (Discovery Hook)
- Product 4: Brow gel (POV Hook)
- Why: Target specific pain points, build authority
- Audience: Engaged beauty enthusiasts

**Day 5-7: Expand + Test**
- Product 5: Skincare tool (Comparison)
- Product 6: Fashion item (Social Proof)
- Product 7: Back to winning category from Day 1-4
- Why: Test adjacencies, double down on winners
- Audience: Growing loyal following

### Hook Rotation Strategy

**Never use same hook format 2 days in a row.**

Optimal Week 1 rotation:
- Monday: Price Reveal
- Tuesday: POV
- Wednesday: Discovery
- Thursday: Price Reveal (different product category)
- Friday: Social Proof
- Saturday: Comparison
- Sunday: Before/After

**Why:** Algorithm reads this as diverse content creator, not one-trick account.

## 🚨 RED FLAGS — When to Pivot

### If Video Underperforms (<1K views in 24hrs):

**Check:**
1. **Hook mismatch?** → Try different format tomorrow
2. **Content too long?** → Cut to 15s next time
3. **CTA unclear?** → Test "comment LINK" strategy
4. **Product boring?** → Switch to viral/trending item

**Pivot decision tree:**
- 3 videos same hook, all underperform → Change hook format
- 3 videos same product type, all underperform → Change category
- 3 videos same setting, all underperform → Change visual aesthetic

---

## ✅ SUCCESS PATTERNS — When to Replicate

### If Video Hits (10K+ views in 6hrs):

**Replicate immediately:**
1. Use EXACT same hook format
2. Use EXACT same character shots (re-use Freepik images)
3. Change only: product, setting details, outfit
4. Post 24 hours later
5. Caption: "Like the [previous product]? You'll love this."

**Example replication:**
- Video 1 (hit): "I spent $15 on this foundation" → 25K views
- Video 2 (replicate): "I spent $9 on this concealer" → Same hook, similar product
- Video 3 (double down): "I spent $12 on this setting spray" → Pattern established

**Result:** By Video 3, you're THE affordable makeup reviewer on TikTok.
