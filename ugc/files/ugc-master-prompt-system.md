## 🎣 SECTION 2: VIRAL HOOK FORMATS (Pick One)

### Format 1: Price Reveal Hook
**Structure:** "I spent $[X] on this [product] and [unexpected outcome]"
**Best for:** Mid-range products ($15-60), Fashion, Beauty
**Example:** "I spent $23 on this serum and threw out my $80 one"

### Format 2: POV Hook
**Structure:** "POV: You finally found [specific problem solution]"
**Best for:** Problem-solving products, relatable struggles
**Example:** "POV: You found jeans that fit your waist AND hips"

### Format 3: Twist Hook
**Structure:** "Everyone told me [not to/to] [action]. They were [wrong/right]."
**Best for:** Contrarian takes, underdog products
**Example:** "Everyone said skip the $15 moisturizer. Big mistake."

### Format 4: Social Proof Hook
**Structure:** "I've seen [X people/places] using this. Here's why."
**Best for:** Trending products, location-based content
**Example:** "3 influencers I follow use this. Here's the real tea."

### Format 5: Opinion Hook
**Structure:** "Hot take: [bold statement about category] — except this one"
**Best for:** Crowded markets, differentiation plays
**Example:** "Most $30 dresses are trash. This one isn't."

### Format 6: Discovery Hook
**Structure:** "Wait, why is no one talking about [product feature]?"
**Best for:** Underrated products, hidden features
**Example:** "Why is no one talking about this built-in SPF?"

### Format 7: Before/After Hook
**Structure:** "I tried this for [X days/weeks]. Here's what changed."
**Best for:** Skincare, supplements, habit-forming products
**Example:** "I used this for 2 weeks. My skin is obsessed."

### Format 8: Comparison Hook
**Structure:** "[Expensive brand] vs [affordable brand]. Let me explain."
**Best for:** Dupes, budget alternatives
**Example:** "Drunk Elephant vs CeraVe. The difference is wild."

## 🎬 SECTION 3: CONTENT FORMATS (Pick One)

### Format A: Product Demo (15-20s)
- **0-2s:** Hook + product reveal
- **2-12s:** Hands-on demonstration, key features
- **12-18s:** Honest reaction/caveat
- **18-20s:** CTA
- **Best for:** Makeup, skincare, tools, gadgets

### Format B: Try-On/Unboxing (20-30s)
- **0-2s:** Hook while opening/holding product
- **2-15s:** Trying on/testing product
- **15-25s:** Fit/result commentary
- **25-30s:** CTA
- **Best for:** Fashion, accessories, shoes

### Format C: Comparison Shot (15-20s)
- **0-2s:** "This vs That" hook
- **2-10s:** Side-by-side visual comparison
- **10-18s:** Key difference explained
- **18-20s:** CTA
- **Best for:** Dupes, before/after, alternatives

### Format D: Storytime UGC (20-30s)
- **0-3s:** Hook setting up story
- **3-20s:** Short narrative arc with product
- **20-28s:** Payoff/conclusion
- **28-30s:** CTA
- **Best for:** Problem-solution products, transformations

### Format E: List Format (20-30s)
- **0-2s:** Hook announcing list
- **2-25s:** 3-5 rapid points with product visible
- **25-30s:** CTA
- **Best for:** Multi-benefit products, versatile items

## 🔄 SECTION 8: CHARACTER CONSISTENCY RULES

### Non-Negotiable Consistency Elements

**ALWAYS IDENTICAL:**
1. **Face structure** - same generated face across all videos
2. **Hair** - exact color, length, style (no variations)
3. **Skin tone** - consistent across all lighting
4. **Age appearance** - no aging up/down between videos
5. **Base style aesthetic** - same photography vibe

**CAN VARY (but stay on-brand):**
1. **Outfit** - changes per setting/product (but same style tier)
2. **Expression** - adapts to hook emotion
3. **Background** - changes per product category
4. **Lighting warmth** - slight variation ok for setting realism
5. **Product interaction** - different poses per product type

## 💡 ADVANCED TIPS

### Subtle Variations to Prevent AI Detection
- Change background props slightly between videos
- Vary character expression naturally (don't use same "smile" every time)
- Use different hand positions when holding products
- Alternate between left/right hand product holding
- Mix lighting temperatures (warm vs cool) across video batch

### Voiceover Authenticity Markers
- Include verbal fillers naturally: "like," "honestly," "literally"
- Use contractions: "it's" not "it is", "don't" not "do not"
- End sentences with slight upward inflection on opinions
- Pause before product reveal (builds anticipation)
- Speed up slightly when excited, slow down for caveats

### Caption Psychology
- Start caption with hook (repeat video hook)
- Use emoji strategically (1-2 max, only if on-brand)
- Break caption into 3 lines max (readability)
- End with CTA + arrow pointing down
- Never use more than 5 hashtags (looks desperate)

### Trending Sound Strategy
- Use sound from "7-day trending" list only
- Avoid sounds with >500K videos (oversaturated)
- Pick sounds with 50K-200K uses (sweet spot)
- Layer at 30% volume (voiceover is hero)
- Never use same sound twice in one week
